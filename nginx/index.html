Nginx

2017-11-13

CentOS安装

1.下载nginx包
2.wget  http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm
3.建立nginx的yum仓库
4.rpm -ivh nginx-release-centos-7-0.el7.ngx.noarch.rpm
5.yum install nginx
6.启动：systemctl start nginx
7.设置开机自启动：systemctl enable nginx.service

静态文件服务器

  • 打开/etc/nginx/conf.d中配置文件 json location / { root /goapp/myBlog; index index.html index.htm; }
  • 修改root指向静态文件

常用命令

  • systemctl start nginx.service
  • systemctl stop nginx.service
