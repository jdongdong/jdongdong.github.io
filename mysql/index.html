Mysql

2017-11-13

Mysql安装

  • yum repository地址：http://dev.mysql.com/downloads/repo/yum/
  • 查看最新rpm Package（mysql57-community-release-el7-11.noarch.rpm）
  • wget http://repo.mysql.com/mysql57-community-release-el7-11.noarch.rpm
  • rpm -ivh mysql57-community-release-el7-11.noarch.rpm
  • yum install mysql-community-server

Mysql配置

启动

  • 开机自启动:systemctl enable mysqld
  • 启动:systemctl start mysqld

密码配置&&开放远程连接

  • 查询自动生成的随机密码：sudo grep ‘temporary password’ /var/log/mysqld.log
  • mysql -uroot -p
  • ALTER USER ‘root’@‘localhost’ IDENTIFIED BY ‘新密码’;新密码必须包含大写、小写字母、数据、特殊字符，大于八位
  • GRANT ALL PRIVILEGES ON . TO ‘root’@‘%’ IDENTIFIED BY ‘密码’ WITH GRANT OPTION;
  • FLUSH PRIVILEGES;
  • exit
